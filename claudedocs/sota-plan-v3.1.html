Cat C

Storage & performance — 5종

데모 머신 부담 ↓, 인덱싱 가속, 인덱스 효율.

TurboQuant bits2 (1.18 NEW · NOT Binary Quantization) SHIP

⚠️ research/10 audit 결과 수정: TurboQuant ≠ Binary Quantization. TurboQuant는 1.18 신규 family ("turbo" config, bits4/bits2/bits1_5/bits1 enum). Binary Quant는 1.15의 별도 family ("binary" config). v3 초안의 "Binary 2-bit" 표기는 잘못. 올바른 syntax:

"quantization_config": {
  "turbo": { "bits": "Bits2", "always_ram": true }
}

bits2로 vector 16× 압축. 6개 벡터 N=10k → storage 2GB → ~125MB.

Q-fit 5/5 demo-fit 1/5 (invisible) Source: research/06 §3.13 · research/10 Verdict 2

TurboQuant rescore + oversampling pairing SHIP

research/10 권고 필수 페어링: bits2 단독은 recall ~10-15% 저하. search_params.quantization.rescore=true + oversampling=2.0로 quantized HNSW에서 후보 수집 후 raw vector로 rescore → recall 회복. 1.18 GA, Rust client 노출.

"params": { "quantization": {
  "ignore": false, "rescore": true, "oversampling": 2.0
} }
Q-fit 4/5 (KC-01 정상화) demo-fit 0/5 (invisible) Effort: ~1h · KC-01에 페어링
